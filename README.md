# siefe.vim
## Vim search plugin
Based on [fzf.vim](https://github.com/junegunn/fzf.vim)

## Commands

| Command           | List                                                                                  |
| ---               | ---                                                                                   |
| `:SiefeRg`        | rg search                                                                             |
| `SiefeRgVisual`   | rg search of the visual selection                                                     |
| `SiefeRgWord`     | rg search of the word under the cursor                                                |
| `SiefeRgWORD`     | rg search of the WORD under the cursor                                                |
